# 행렬 2차원 배열

```python
## 상/하/좌/우 이동
dx, dy = (-1, 1, 0, 0), (0, 0, -1, 1)

## 행의 개수 N, 열의 개수 M
N, M = map(int, input().split())
G = [list(map(int, input().split())) for _ in range(N)]
visited = [[0 for _ in range(M)] for _ in range(N)]
```

## 순차 탐색 (방문한 노드 출력) - DFS 또는 BFS

```python
def dfs(x, y):
    visited[x][y] = 1
    for k in range(4):
        nx, ny = x + dx[k], y + dy[k]
        if 0 <= nx < N and 0 <= ny < M:
            if not visited[nx][ny]:
                dfs(nx, ny)
```

```python
def bfs(x0, y0):
    visited[x0][y0] = 1
    q = deque([(x0, y0)])
    while q:
        x, y = q.popleft()
        for k in range(4):
            nx, ny = x + dx[k], y + dy[k]
            if 0 <= nx < N and 0 <= ny < M:
                if not visited[nx][ny]:
                    visited[nx][ny] = 1
                    q.append((nx, ny))
```
